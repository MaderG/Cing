# Cing

Este é um projeto feito para a cadeira de Introdução a Multimídia. É uma versão do jogo inspirada no Flappy Bird, porém, o pássaro voa de acordo com a frequência da voz do jogador, em vez de cliques ou toques.

## Instalação

Este projeto requer as seguintes bibliotecas Python:

- pyaudio
- numpy
- scipy
- pygame

Você pode instalar todas as dependências com o seguinte comando:

```bash
pip install pyaudio numpy scipy pygame
```

## Execução
Depois de instalar todas as dependências necessárias, você pode executar o jogo a partir da linha de comando. Certifique-se de que você está no diretório correto onde o arquivo game.py está localizado.

Para iniciar o jogo, use o seguinte comando:

```bash
python game.py
```

Este comando inicia o jogo. Lembre-se, a frequência da sua voz controla o voo do pássaro!
